# STM32 AI Flow

STM32 AI Flow is a documentation-first public surface for a reusable STM32
workflow that is currently being refined through real project work and active
testing.

This public docs repository now mirrors the canonical user-facing
documentation from the working STM32 AI Flow starter kit as closely as
practical.

The working starter kit and some operational or workflow-owned surfaces are
still under active development and are not yet broadly published here as a
general public package.

If you are interested in early access, would like to evaluate the workflow for
a real STM32 project, or would be happy to participate as an early adopter or
tester, please open a GitHub Discussion in this repository. Access may be
provided on request.

The Markdown files are the primary public-facing source in this docs set.
PDF and HTML reading copies are provided for convenience.

What this public docs set includes:

- repository license and notice files
- Markdown guides for onboarding and evaluation
- HTML and PDF reading copies
- diagrams and image assets used by those guides

What it does not include:

- workflow scripts
- templates
- skills
- maintainer policy
- the reusable starter implementation itself

If a document mentions a workflow-owned file or operational surface that is not
present here, treat that as a reference to the working starter kit rather than
as a broken concept. The intention is to keep the public docs close to the real
starter documentation without pretending that the full working kit is already
published.
