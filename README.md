# STM32 AI Flow

STM32 AI Flow is a documentation-first public surface for a reusable STM32
workflow that is currently being refined through real project work and active
testing.

For now, this public repository is intentionally limited to user-facing
documentation, diagrams, and reading copies. The reusable starter kit and its
implementation are still being hardened and are not yet broadly published as a
general public package.

Sorry for the extra friction at this stage. The goal is to share something
solid and genuinely helpful rather than exposing the workflow kit too early
while important parts are still evolving.

If you are interested in early access, would like to evaluate the workflow for
a real STM32 project, or would be happy to participate as an early adopter or
tester, access may be provided on request.

- `docs/pdf/workflow-safety.pdf`
- `docs/pdf/agent-communication-guide.pdf`

The Markdown files are the primary public-facing source in this docs set.
PDF reading copies are provided for convenience and may occasionally lag
slightly behind the latest Markdown wording while the public-doc export path is
still being refined.

What this public docs set includes:

- Markdown guides for onboarding and evaluation
- diagrams and image assets used by those guides
- PDF reading copies

What it does not include:

- workflow scripts
- templates
- skills
- maintainer policy
- the reusable starter implementation itself
