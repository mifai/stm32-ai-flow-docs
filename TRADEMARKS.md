# Trademarks

This file records mark-use guidance for `STM32 AI Flow`.

## Project Name

`STM32 AI Flow` identifies this project and its associated workflow materials.
The Apache-2.0 software license in this repository does not grant trademark
rights beyond reasonable nominative use.

Reasonable nominative use includes:

- saying that a repo, fork, or derivative is based on or derived from
  `STM32 AI Flow`
- linking to this repository
- accurately describing compatibility with this workflow

Use that should be avoided without separate permission includes:

- implying endorsement by the original project maintainers
- presenting a modified fork as if it were the canonical `STM32 AI Flow`
- using the project name or any future project logos as primary branding for an
  unrelated product

If you redistribute a materially modified fork, prefer one of these patterns:

- rename the fork, or
- keep the original name only as an attribution line such as
  `Based on STM32 AI Flow`
